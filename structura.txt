ASOSIY G'OYA
─────────────
ML modellarini o'zimiz qurmayapmiz.
Buning o'rniga:
  1. Bank tranzaksion ma'lumotlari (PostgreSQL — MCP Server orqali)
  2. Matematik/statistik algoritmlar (sof Python funksiyalari)
  3. AI Agent (Claude tool_use) — qaysi modelni ishlatishni o'zi hal qiladi,
     MCP tools orqali DB dan ma'lumot oladi, algoritmlarni chaqiradi,
     yakuniy Ha/Yo'q qarorini beradi

MVP UCHUN 5 TA MODEL
─────────────────────
  M-A3  Saturation Index      — nisha qanchalik to'yingan? (0–100)
  M-C1  Location Score        — lokatsiya qanchalik jozibali? (0–100)
  M-D1  Viability Check       — biznes omon qolishiga ehtimol (Monte Carlo)
  M-D3  ROI Estimator         — NPV, ROI, qaytish muddati (DCF)
  M-E2  Churn Prediction      — 2 yilda yopilish ehtimoli (%)

AGENT ISHLASH JARAYONI (AGENT LOOP)
═══════════════════════════════════════════════════════════════════

1. Foydalanuvchi so'rovi keladi:
   { nisha: "Kafe", koordinata: [41.299, 69.240], kapital: 50_000_000 }

2. AgentOrchestrator Claude ga system prompt + user so'rovini yuboradi.
   System prompt prompt caching bilan — bank qoidalari, 5 model tavsifi.

3. Claude tool_use orqali qadamma-qadam ishlaydi:

   Qadam 1 → MCP tool: get_niche_transactions("Kafe", radius=1000)
   Qadam 2 → MCP tool: get_competitors(41.299, 69.240, radius=500)
   Qadam 3 → MCP tool: get_location_poi(41.299, 69.240)
   Qadam 4 → MCP tool: get_sector_benchmarks("Kafe")

   Qadam 5 → Algorithm tool: run_saturation_index(tranzaksiyalar, raqobatchilar)
             ← { score: 72, status: "yuqori to'yinganlik" }

   Qadam 6 → Algorithm tool: run_location_score(poi_data, trafik)
             ← { score: 68, sub_scores: {...} }

   Qadam 7 → Algorithm tool: run_viability_check(kapital, benchmarks)
             ← { survival_2y: 0.61, breakeven_months: 8, runway_months: 14 }

   Qadam 8 → Algorithm tool: run_roi_estimator(kapital, cashflow)
             ← { npv: 12_400_000, roi: 0.248, payback_months: 18 }

   Qadam 9 → Algorithm tool: run_churn_prediction(location_score, saturation)
             ← { closure_probability: 0.38 }

4. Claude barcha natijalarni umumlashtiradi:
   → Yakuniy qaror: "YO'Q — nisha yuqori to'yingan (72/100),
     yopilish ehtimoli 38%. ROI ijobiy lekin margin past."

5. AnalysisResponse JSON — dashboard uchun
